# Portfolio Website

This is a personal portfolio website built using [Create React App](https://create-react-app.dev/). It showcases my skills, projects, experience, and contact information.

## 🚀 Features

- 🖥️ Responsive design for desktop, tablet, and mobile
- 🧑 About section with bio and photo
- 💼 Projects showcase with live links and descriptions
- 🛠️ Skills section with categorized technology list
- 📄 Resume download
- 📬 Contact form with email integration

## 🛠️ Tech Stack

- **Frontend:** React, JSX, CSS (or styled-components)
- **Tooling:** Create React App, React Router
- **Deployment:** Vercel
